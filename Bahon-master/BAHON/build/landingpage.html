What is Bahon?



The project focuses on developing an Iot and Embedded system based software for Smart Transportation. The main objective of the project is to automate several processes of public transportation and enhance the passenger experience. Each passenger will have a smart card that they have to scan entering and exiting the bus. The system will automatically calculate the travel distance and corresponding fare which will be deducted from their account at the point of exit. If a passenger forgets to carry a Smart card he/she can scan a temporary QR code generated from the app. The project will feature a user app and an admin panel. The user app will offer functionalities such as travel history, balance checking and recharging, journey planning, fare calculation, notifications, profile setting, and report issues. The admin panel will include a dashboard, revenue reports, user management, journey data analysis, fare adjustment and maintenance. For implementation of the software, We use an object-oriented software architecture. Key components will include classes for passengers, cards, entry and exit points, journeys, fare calculation, and overall system control. The software will handle real-time fare calculation, balance updates, and travel history. For the implementation hardware, we will use smart card readers, microcontrollers like Arduino for processing data, GPS and IOT devices. Bypass solution in our project ensures seamless operation and user convenience in various scenarios. If a passenger forgets their smart card, they can generate a temporary QR code through our mobile app for scanning at entry and exit points. For forgotten exit scans, the system calculates fares based on estimated maximum distance.

Key Features

Admin Portal

Manage and oversee the entire transportation system with ease using the admin portal. It allows administrators to configure settings, monitor real-time data, and manage user access.

Dashboard

Get an overview of all key metrics and insights in a clean, intuitive dashboard. Visualize system performance, usage statistics, and operational trends with powerful data visualization tools.

Revenue Reports

Track and analyze revenue from ticket sales, subscriptions, and other streams. Generate reports to understand financial trends and optimize pricing strategies.

User Management

Efficiently manage users, including drivers, passengers, and administrators. Create user profiles, assign roles, and handle permissions from one centralized location.

Journey Data

Monitor and analyze journey data in real-time, including routes, distances, and travel times. Use this data for optimization and planning of transportation routes and schedules.

Fare Adjustment

Set and adjust fares dynamically based on demand, route, or time of day. Implement promotions, discounts, and flexible pricing strategies to improve user engagement.

Maintenance Scheduling

Schedule and track maintenance activities for vehicles and infrastructure. Keep the fleet in optimal condition and minimize downtime by automating routine tasks.

Issue Reporting

Allow users and staff to report issues easily, from broken vehicles to operational delays. Track progress and resolution to ensure efficient problem-solving and system reliability.

Technologies We Use

Bahon combines advanced hardware and cloud technologies to deliver seamless performance:

  • ESP32 Microcontroller: Manages hardware and server communication.
  • GPS Module: Provides real-time location tracking.
  • RFID/NFC Scanner: Enables cashless entry and exit tracking.
  • Firebase Realtime Database: Ensures smooth data synchronization.
  • Node.js: Powers the backend logic.
